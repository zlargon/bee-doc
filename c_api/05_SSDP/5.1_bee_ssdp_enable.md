## 5-1. bee_ssdp_enable

```c
int bee_ssdp_enable(bool option);
```

### Description

enable/disable SSDP

* default option: `true`

### Parameters

| NO | NAME | TYPE | DESCRIPTION |
| :---: | --- | --- | --- |
| 1 | **option** | `bool` | `ture` : enable SSDP<br> `false` : disable SSDP |


### Return Value

| TYPE | VALUE | DESCRIPTION |
| :---: | :---: | --- |
| `int` | 0 | success (always) |

### Support Platform

| iOS | Device |
| --- | --- |
| `YES` | `YES` |
