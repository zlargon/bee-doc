## 3-3. bee_get_token

```c
const char * bee_get_token();
```

### Description

Get user/device's `token`.

* `token` is used to access Cloud HTTPs API
* call this function after user/device login success

### Return Value

| TYPE | VALUE |
| :--: | :--: |
| `const char *` | &lt;token&gt; |

### Support Platform

| iOS | Device |
| -- | -- |
| `YES` | `YES` |
