## 3-2. bee_get_uid

```c
const char * bee_get_uid();
```

### Description

Get user/device's cloud id.

* `uid` is a unique id in Cloud Service.
* call this function after user/device login success

### Return Value

| TYPE | VALUE |
| :--: | :--: |
| `const char *` | &lt;uid&gt; |

### Support Platform

| iOS | Device |
| -- | -- |
| `YES` | `YES` |
