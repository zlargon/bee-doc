## 4-2. ssdpDebug

```java
public void ssdpDebug(boolean option);
```

### Description

enable/disable SSDP log.

* default option: `false`

### Parameters

| NO | NAME | TYPE | DESCRIPTION |
| :---: | --- | --- | --- |
| 1 | **option** | `boolean` | `ture` : enable SSDP debug log<br> `false` : disable SSDP debug log |
